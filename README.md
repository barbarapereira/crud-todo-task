# crud-todo-task# TodoList - TypeScript

Projeto de CRUD de tarefas para praticar fundamentos de TypeScript.

## Funcionalidades planejadas:
- [ ] Criar tarefa
- [ ] Listar tarefas
- [ ] Editar tarefa
- [ ] Deletar tarefa
- [ ] Marcar status (não iniciada, em progresso, concluída)
- [ ] Filtrar por status
- [ ] Buscar tarefas
- [ ] Ordenar tarefas
- [ ] Paginação

## Tecnologias:
- TypeScript
- Node.js
- Jest (testes)
